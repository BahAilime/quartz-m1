# Définition

Une **épreuve ou expérience aléatoire** est un processus ou un événement dont le résultat **ne peut pas être prédit avec certitude** à l'avance, bien qu'on sache les **possibilités** de résultats.

> *Exemple : Lancer un dé, tirer une carte d'un jeu, observer le sexe d’un nouveau-né.*

# Caractéristiques clés

1. **Indéterminisme** : Le résultat n’est pas prévisible avec certitude.
2. **Résultats possibles** : On connaît l'ensemble des résultats possibles (l'**espace échantillon**).
3. **Répétabilité** : L'expérience peut être répétée sous des conditions similaires.
4. **Probabilités** : On peut attribuer une probabilité à chaque résultat possible.

# Exemple concret : Lancer un dé

- **Expérience aléatoire** : Lancer un dé à 6 faces.
- **Espace échantillon** : $S = \{1, 2, 3, 4, 5, 6\}$
- **Résultat possible** : Un nombre entre 1 et 6.
- **Probabilité** : Chaque face a une probabilité de $\frac{1}{6}$.

# En résumé

| Concept | Définition |
|--------|------------|
| **Épreuve aléatoire** | Expérience dont le résultat est incertain |
| **Espace échantillon** | Ensemble de tous les résultats possibles |
| **Événement** | Sous-ensemble de l'espace échantillon |
| **Probabilité** | Mesure de la chance qu’un événement se produise |
