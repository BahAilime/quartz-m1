# Définition
L'**écart type** est une mesure de **dispersion** d'un ensemble de données. Il indique la **distance moyenne** entre chaque valeur et la **moyenne**.
L'écart type est **en unités de la variable** donc facile à interpréter !

# Formule

L'écart type est la racine carrée de la [[variance]], c'est à dire:

$$\sigma = \sqrt{\frac{1}{N} \sum_{i=1}^{N} (x_i - \mu)^2}$$

- $\sigma$ : écart type  
- $N$ : nombre total d'observations  
- $\mu$ : moyenne  
- $x_i$ : chaque valeur

# Interprétation
- Plus l'écart type est **grand**, plus les données sont **dispersées**.  
- Plus il est **petit**, plus les données sont **groupées** autour de la moyenne.

# Exemple
Données : [2, 4, 4, 4, 5]  
→ Moyenne = 3.8  
→ Écart type ≈ 0.8 → données proches de la moyenne.

## 💡 Astuce
L'écart type est **en unités de la variable** → facile à interpréter !
