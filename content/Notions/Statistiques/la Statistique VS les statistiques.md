
# 1. La Statistique : la science des données

La **statistique** est une **discipline scientifique** qui vise à :

- Collecter des données
- Les organiser
- Les analyser
- En tirer des conclusions fiables
- Évaluer les incertitudes

C’est une **méthode rigoureuse**, fondée sur des principes mathématiques, des hypothèses, des tests d’hypothèses, des modèles probabilistes.

> Exemple : Un chercheur utilise la statistique pour déterminer si un nouveau médicament est efficace en comparant les résultats d’un groupe expérimental et d’un groupe contrôle.

# 2. Les statistiques : les faits ou les chiffres

**Les statistiques** désignent les **données numériques** ou les **faits** rapportés, souvent dans des contextes médiatiques ou politiques.

> Exemple : "Le taux d’emploi est de 72 % en 2024."

👉 Ce chiffre est **un fait**, une observation, mais **ne dit pas** comment il a été obtenu, ni si il est fiable, ni si il est représentatif.

## ⚠️ Les pièges fréquents

| Erreur                                                | Explication                                                                                         |
| ----------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| **Confondre les données avec des conclusions**        | Dire "les données montrent que X est vrai" sans justifier la méthode.                               |
| **Utiliser des statistiques sans contexte**           | Un chiffre isolé peut être trompeur (ex : taux de chômage de 2 % dans une ville de 1000 habitants). |
| **Ignorer les biais**                                 | [[Biais de sélection]], [[biais de rappel]], [[biais de confirmation]]                              |
| **Interpréter les statistiques comme des certitudes** | Les statistiques reflètent des probabilités, pas des certitudes absolues.                           |
